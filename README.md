## Grid
[Official guide](https://getuikit.com/docs/grid)

~~~javascript
import {Grid} from 'react-uikit3';

React.render((
  <Grid>
    <div className={'uk-width-1-2'}>
      Left column
    </div>
    <div className={'uk-width-1-2'}>
      Right column
    </div>
  </Grid>
), element)
~~~

Attribute |  Type  | Required?
----------|--------|---------
className | string | false
